# Offer 智选 - Offer Wise

大数据和人工智能助你选出最适合你的 Offer。

## 主要功能 💡

- 接受 个人及 Offer 的关键信息的输入
- 利用大数据和人工智能，给你的 Offer 进行评估
- 给出 Offer 的综合评分及选择建议
